## Message spine: live vs. approved

The approved spine includes "mapped operating zones." Every Operational Precision buyer page uses "360-degree sensing" instead. This has become the de facto site spine.

**Live formula (all buyer pages):**
> "Snowbotix automates hard-to-staff outdoor maintenance routes with supervised autonomy, **360-degree sensing**, human-in-the-loop monitoring, and proof-of-service records."

| Element | Approved spine | Live formula | Match? |
|---|---|---|---|
| "hard-to-staff outdoor maintenance routes" | ✓ | ✓ | Yes |
| "supervised autonomy" | ✓ | ✓ | Yes |
| "mapped operating zones" | ✓ | ✗ | **No — replaced by "360-degree sensing"** |
| "human-in-the-loop monitoring" | ✓ | ✓ | Yes |
| "proof-of-service records" | ✓ | ✓ | Yes |

**Decision needed:** Choose one. Both are true and defensible. "Mapped operating zones" answers "is this safe and bounded?" — the buyer's trust question. "360-degree sensing" answers "how does it avoid people?" — the mechanics question. Recommend combining: "supervised autonomy, mapped operating zones and 360-degree sensing, human-in-the-loop monitoring, and proof-of-service records."

## Audience routing map

The site serves three distinct audiences with two different design systems. This table shows where each audience should be routed and whether the current pages support that path.

| Audience | Entry point | Path | Design system | Problem |
|---|---|---|---|---|
| **Buyer (facilities, operations, property)** | Homepage | → Solutions / Who It's For → Solution page → How It Works → CTA | Should be Operational Precision throughout | Homepage is Industrial Precision |
| **Investor** | Investors page (direct link) | → Company → Team | Industrial Precision | Correct |
| **Candidate** | Careers (direct link or search) | → Company → Team | Industrial Precision (dark) | Correct |
| **Press / partner** | Company page | → Team → Investors | Industrial Precision | Correct |

**The buyer routing problem:** The homepage is the primary SEO and paid traffic entry point. It uses Industrial Precision. Buyers who enter here and click any solution link encounter Operational Precision. The visual shift is disorienting and signals "you've left the brand" when in fact you've entered the buyer journey.
